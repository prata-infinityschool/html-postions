Exemplos de CSS Position

Position: Static

Este é o valor padrão. Os elementos são posicionados de acordo com o fluxo normal da página.

Static

Position: Relative

Os elementos são posicionados em relação à sua posição original no fluxo normal da página.

Relative

Position: Absolute

Os elementos são posicionados em relação ao seu contêiner mais próximo com `position: relative`, `absolute` ou `fixed`.

Absolute

Position: Fixed

Os elementos são posicionados em relação à janela de visualização e não se movem quando a página é rolada.

Fixed

Position: Sticky

Os elementos se comportam como `relative` até que rolem para um ponto especificado, onde então se comportam como `fixed`.

Sticky

Role a página para ver o comportamento do elemento sticky.
